# Project Agents

This directory is for project-specific Claude Code agents.

## Installation

Copy agents from the template library:

```powershell
# Copy all agents
Copy-Item E:\github\claude_code_baseline\agents\*.md .claude\agents
# Or copy specific agent
Copy-Item E:\github\claude_code_baseline\agents\gen-docs.md .claude\agents```

## Available Agent Templates

See: `E:\github\claude_code_baseline\agents\README.md`
